## 📖 **Описание**

Бот помогает клиентам записываться на маникюрные услуги, а администратору — управлять записями.

**Клиенты могут:**
- Записаться на услугу (Маникюр, Педикюр, Дизайн, Наращивание)
- Просматривать свои записи
- Отменять записи

**Администратор может:**
- Просматривать все записи
- Получать PDF-отчёты на 3 дня
- Получать уведомления о новых записях

---

## ✨ **Возможности**

| Функция | Описание |
|---------|----------|
| 📅 Запись | Пошаговая запись на услугу (услуга → дата → время → контакт → подтверждение) |
| 📋 Мои записи | Просмотр активных записей |
| ❌ Отмена записи | Отмена существующей записи |
| 📊 Отчёт | PDF-отчёт с записями на 3 дня (только для админа) |
| 📋 Все записи | Просмотр всех активных записей (только для админа) |
| 🔔 Напоминания | Автоматические напоминания о записи (каждый день в 9:00) |

## 🛠 **Технологии**

- **Go** 1.24+
- **PostgreSQL** (хранение данных)
- **Docker** (контейнеризация)
- **Telegram Bot API** (интерфейс)
- **GORM** (ORM для работы с БД)
- **Cron** (планировщик напоминаний)
